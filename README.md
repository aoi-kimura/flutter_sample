# flutter_sample

アプリによくありそうな基本的な機能をFlutter(MVVM)で実装するサンプル

## 主な機能・コンポーネント

* リスト表示
* TextField
* API通信(GET)
* 画面遷移

## 使用してるライブラリ

### json_annotation

`/model`配下編集したらルートディレクトリで以下コマンド実行する

```
flutter packages pub run build_runner build
```
